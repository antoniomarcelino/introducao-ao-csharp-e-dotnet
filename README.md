## Primeiros Passos

Para colocar uma cópia local em execução, siga estas etapas simples.

### Pre-requisitos

* [Visual Studio Community](https://visualstudio.microsoft.com/pt-br/downloads/)
* [.NET Core](https://learn.microsoft.com/pt-br/dotnet/core/install/windows?tabs=net60#install-with-visual-studio)
* [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)

### Como executar esse projeto?

* ### Opção 1: Descarregar localmente
1. No [repo](https://github.com/antoniomarcelino/introducao-ao-csharp-e-dotnet.git) do GitHub, clique no botão verde "Code"
2. Selecione a opção "Download ZIP"
3. Abra a solução (arquivo .sln) pelo Visual Studio
4. Apenas clique no botão "Iniciar Depuração" (ou pressione F5) e o Visual Studio apresentará a aplicação no console

* ### Opção 2: Clonar o projeto

* #### 2.1: Clonar pelo GitHub

1. No [repo](https://github.com/antoniomarcelino/introducao-ao-csharp-e-dotnet.git) do GitHub, clique no botão verde "Code"
2. Selecione a opção "Open with Visual Studio" (As informações do repo aparecerão numa nova janela do Visual Studio)
3. Informe o caminho do seu repositório local e clique no botão "Clonar"
4. Apenas clique no botão "Iniciar Depuração" (ou pressione F5) e o Visual Studio apresentará a aplicação no console

* #### 2.2: Clonar pelo Visual Studio

1. Na barra superior do Visual Studio, acesse Git > Clonar o repositório...
3. Informe a URL do repo do GitHub
```
https://github.com/antoniomarcelino/introducao-ao-csharp-e-dotnet.git
```
4. Informe o caminho do seu repositório local e clique no botão "Clonar"
5. Apenas clique no botão "Iniciar Depuração" (ou pressione F5) e o Visual Studio apresentará a aplicação no console
 ##### Para mais informações, acessar [Como clonar um repositório no Visual Studio](https://learn.microsoft.com/pt-br/visualstudio/version-control/git-clone-repository?view=vs-2022)
